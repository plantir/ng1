# Cashineh Front-End application

1.  angularjs 1.7.2
2.  typescript 3
3.  webpack 4
4.  angular material
5.  bootstrap 4
6.  flex box
7.  scss
8.  component base
9.  ssr
10. lazyLoad

## Get the project

```bash
  git pull origin master
```

## Setup

copy from .env.example to .env file and change content

run npm install

```bash
  npm install
```

## start Project for Dev

```bash
    npm start
```

## build project

```bash
    npm run build
```

## serve project 

after build command you must run serve command for serve project in ssr mode

```bash
    npm run serve
```
